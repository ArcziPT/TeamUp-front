Team Up
How to use it?

1. Create Account

When creating new account you will be asked to create your profile description and select the skills, which you have.


2.a Find a Project

If you want to contribute to other projects you can search for them and if they have active job offers you can apply and be accepted into the project or you can be personally invited by the administrators of one.


2.b Create a Project

If you have great idea, but you don't have enough teammates, you can find them here. Just create the new project and add suitable job postings or search for specified users.
